### Mi-V Soft CPUs
Mi-V Ecosystem supports following RV32I base instruction set processors with optional extensions.
- MIV_RV32
- MIV_RV32IMA_L1_AHB
- MIV_RV32IMA_L1_AXI
- MIV_RV32IMAF_L1_AHB
